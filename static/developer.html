📖 API Overview

The OpenGradient API provides programmatic access to our AI model hub. Build applications that search, compare, and interact with 2,000+ AI models.

Base URL

https://og-model-assistant.onrender.com/api

All API requests should be prefixed with this base URL.

Key Features

  • 🔍 Smart Search — Find models using natural language
  • 💬 AI Chat — Interact with models via chat interface
  • 📊 Analytics — Track usage and statistics
  • 🎯 Model Status — Real-time availability monitoring
  • 💎 Token System — OPG token integration for payments

📚 API Reference

Complete reference for all available endpoints.

POST /models/search

Search for AI models using natural language queries.

Parameters

query string Natural language search query (e.g., "language models for text generation")
limit integer Maximum number of results to return (default: 10) 
curl -X POST https://og-model-assistant.onrender.com/api/models/search \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "language models", "limit": 5}'
POST /chat

Interact with AI models using chat interface.

Parameters

message string User message to send to the AI
model string Model name to use (e.g., "GROK-4-Fast") 
curl -X POST https://og-model-assistant.onrender.com/api/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!", "model": "GROK-4-Fast"}'
GET /models

Get list of all available AI models.

Query Parameters

limit integer Number of results (default: 10, max: 100)
offset integer Pagination offset 
curl https://og-model-assistant.onrender.com/api/models?limit=10 \
  -H "Authorization: Bearer YOUR_API_KEY"
GET /analytics/stats

Get platform statistics and analytics.

curl https://og-model-assistant.onrender.com/api/analytics/stats \
  -H "Authorization: Bearer YOUR_API_KEY"

⚡ Rate Limits

API rate limits ensure fair usage for all developers.

Default Limits

Endpoint Limit Window
/models/search 100 requests per minute
/chat 60 requests per minute
/models 200 requests per minute
/analytics/* 30 requests per minute

Rate Limit Headers

Every response includes headers with your current rate limit status: 

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1640000000

Handling Rate Limits

When you exceed the rate limit, you'll receive a 429 Too Many Requests response: 

{
  "error": "rate_limit_exceeded",
  "message": "Too many requests. Please retry after 60 seconds.",
  "retry_after": 60
}
