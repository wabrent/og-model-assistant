📚 API Endpoints

Base URL: https://og-model-assistant.onrender.com/api

POST /api/models/search

Search for AI models using natural language queries.

curl -X POST https://og-model-assistant.onrender.com/api/models/search \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "language models", "limit": 5}'
POST /api/chat

Interact with AI models using chat interface.

curl -X POST https://og-model-assistant.onrender.com/api/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!", "model": "GROK-4-Fast"}'
GET /api/models?limit=10&offset=0

Get list of all available AI models with pagination.

curl https://og-model-assistant.onrender.com/api/models?limit=10 \
  -H "Authorization: Bearer YOUR_API_KEY"
GET /api/analytics/stats

Get platform statistics and analytics.

curl https://og-model-assistant.onrender.com/api/analytics/stats \
  -H "Authorization: Bearer YOUR_API_KEY"

📦 SDKs & Libraries

Official SDKs for popular programming languages. Get started quickly with pre-built clients.

🐍

Python SDK

Official Python client with async support and type hints.

pip install opengradient-sdk 
from opengradient import OGClient

client = OGClient(api_key="your_key")
models = await client.search("language models")

for model in models:
    print(model.name, model.description)
🟨

JavaScript SDK

Node.js and browser-compatible client with Promise support.

npm install @opengradient/sdk 
import { OGClient } from '@opengradient/sdk'

const client = new OGClient('your_key')
const models = await client.search('language models')

models.forEach(model => console.log(model.name))
